# steph_pyladies
Repo to store code used for the pyladies event Nov 2023

This script has a bunch of functions which collect data from the clinicaltrials.gov api, clean and aggregate the data and present it as a graph. 

The search condition can be changed by toggling the comment on and off of the conditions identified. 
'polycystic ovary syndrome','endometriosis','menopause','adenomyosis'
